API Tester
Base URL: http://localhost:3000  ·  Token auto-attached to all protected routes
POST /api/auth/register Create account ▾
POST /api/auth/login Login → saves token ▾
POST /api/auth/refresh Get new access token using refresh token ▾

Uses the refresh token saved from login automatically. Run this after login to get a fresh access token.

POST /api/auth/logout Revoke refresh token · 🔒 protected ▾

Revokes the current refresh token in the database and clears all saved tokens from localStorage.

POST /api/auth/refresh (revoked) Must return 401 after logout ▾

Paste the refresh token you used BEFORE logging out. This must return 401. If it returns a new token, logout is broken.
