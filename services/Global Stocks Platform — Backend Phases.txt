Global Stocks Platform — Backend Phases

Phase 0 — Foundation (Done)
- PostgreSQL schema: 9 core tables (users, stocks, portfolios, holdings, orders, trades, wallet_transactions, market_data, watchlists)
- Seed data: initial global stocks across NSE, NYSE, SGX, LSE
- Node.js + Express project skeleton
- Supabase connection via node-postgres

Phase 1 — Core Trading MVP (Done)
- Auth: register, login, JWT access tokens (15m)
- Orders: market + limit, buy + sell
- Trades: execution records linked to orders
- Holdings: quantity + avg cost maintenance
- Wallet: deposit + balance
- Watchlist: add/remove/list
- Basic portfolio P&L (single currency)

Phase 2 — Production-Ready Backend (Current)
- Refresh tokens: 7d, hashed & stored in refresh_tokens
- Auth endpoints: /api/auth/me, /refresh, /logout
- Wallet: withdrawal with balance locking + immutable wallet_transactions
- Stocks: search + filter + pagination on /api/stocks
- Market data: 30 days seeded OHLCV per stock
- Multi-currency P&L: INR/USD/SGD/GBP conversions in portfolio
- Indexes + updated_at triggers for stocks and holdings
- PriceSync service: Yahoo Finance polling every 10s → current_price + market_data

Phase 3 — Frontend MVP
- Next.js 14 App Router app
- Auth pages: login/register
- Screens:
  - Stocks list + search/filter
  - Stock detail with chart
  - Portfolio with P&L
  - Orders history
  - Wallet (deposit/withdraw)
  - Watchlist
- Integration with backend via REST (RTK Query)

Phase 4 — Real-Time UX
- WebSockets channel for live price updates
- Client-side subscription per symbol / portfolio
- Incremental UI updates without polling
- Optimistic updates for orders & wallet

Phase 5 — Hardening for Production
- Redis cache for hot price + portfolio data
- Rate limiting and basic WAF rules
- Structured logging and error tracking
- Health checks and readiness probes
- Background jobs for batch tasks (snapshots, cleanups)

Phase 6 — Advanced Features
- Price alerts (alert rules + history)
- Fundamentals and corporate actions (dividends, splits)
- Indices, benchmarks, and index-style portfolios
- News feed + stock tagging
- KYC, profiles, multi-device sessions
- Deeper analytics: portfolio snapshots, factor breakdowns
